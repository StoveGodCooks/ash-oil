## OUTFIT DESIGN DETAILS

### What Is An "Outfit"?

**Definition:** A single armor piece that represents a complete outfit (helm + chest + feet combined). Provides only stat bonuses, no special effects.

**Why Outfits?**
- Lieutenants feel like a unified squad (matching outfit aesthetic)
- Simpler than modular pieces (1 slot instead of 3)
- Still customizable via 2 accessories
- Easier to balance (single stat package)

**Outfit Mechanics:**
- Equip one outfit per lieutenant
- Can swap at hub before missions
- Cannot change mid-combat
- Stat bonuses stack with accessories + weapon

**Example Loadout - Marcus:**
```
Weapon: Poisoned Dagger (+2 Damage, apply Poison 1)
Outfit: Assassin's Leathers (+1 Speed, +4 HP, +1 Poison damage)
Accessory 1: Ring of Swiftness (+2 Speed bonus)
Accessory 2: Vial of Focus (+1 Mana start)

Result: Marcus is fast, mobile, Poison-focused lieutenant
```
